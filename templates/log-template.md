# Learning Log Template

Append to `logs.md`. Translate headings to the session teaching language.

```md
## <date/time>

### 本次学习内容

- <what was learned>
- <important concepts explained>

### 用户提出的要求

- <teaching preference changes>
- <content requests>

### 掌握情况

- <item>: <mastery assessment>
- <item>: <doubts or fluency>

### 练习情况

- 题目：<brief prompt or type>
- 用户答案：<brief answer or summary>
- 结果：<correct/incorrect/skipped>
- 说明：<misconception or explanation>

### 保留问题 / 待复习

- <unresolved question>

### 下次继续

<next recommended step>
```

## Logging Rules

- Append at the end of a learning session, when the user asks to summarize, or when stopping for now.
- Also append or update when a meaningful misconception appears.
- Record wrong answers clearly enough to review later, but do not be verbose.
- Keep `state.json.resumeSummary` consistent with the latest log.
